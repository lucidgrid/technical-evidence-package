# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) version 2.1.

## Our Standards

- Use welcoming and inclusive language.
- Be respectful of differing viewpoints and experience.
- Accept constructive criticism gracefully.
- Focus on what is best for the project and the community it serves.

## Scope

This code of conduct applies within all project spaces — GitHub issues, pull requests, discussions, and any other project communication channels.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers at **hello@lucidgrid.tech**. All reports will be reviewed and investigated. Maintainers are obligated to maintain confidentiality with regard to the reporter.
